# Удаление старых запретов

1. Зайти в папку с запретом
2. запустить файл service.bat
3. Выбрать пункт 2. Remove Services(Ввести 2 и нажать Enter)
4. Удалить всю папку запрета
5. Скопировать папку zapret-discord-youtube-1.9.7 из архива
6. Запустить файл host.bat из арихва от имени Администратора
